== S017: 多 Writer 同文件冲突处理 ==

merge_writers() lines 348-356:
  if git merge "$BRANCH" --squash --no-edit; then
    git commit ...
    echo "✓ ${TASK_NAME} merged"
    ((MERGED++))
  else
    git merge --abort
    echo "✗ ${TASK_NAME} conflict, deferred"
    ((FAILED++))
  fi

- 第一个 writer merge 成功 ✓
- 第二个 writer 冲突 → merge --abort + 标记 failed ✓
- ((FAILED++)) || true 防止 set -e 崩溃 ✓
